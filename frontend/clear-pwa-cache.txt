清除 PWA 緩存並重新安裝
================================

如果 Install Prompt 沒有出現，可能是因為：
1. Chrome 認為 PWA 已經安裝過
2. Service Worker 緩存了舊狀態
3. 瀏覽器策略限制

解決步驟：
----------

方法 A: 清除 Site Data（推薦）
------------------------------
1. 訪問：https://192.168.10.100:3000

2. 按 F12 打開開發者工具

3. 進入 Application 標籤

4. 左側找到 "Storage" 部分

5. 點擊 "Clear site data"

6. 確認刪除

7. 關閉開發者工具

8. 重新載入頁面（Ctrl + Shift + R 強制刷新）

9. 等待幾秒鐘，應該會看到安裝提示


方法 B: Unregister Service Worker
----------------------------------
1. F12 → Application → Service Workers

2. 找到 https://192.168.10.100:3000

3. 點擊 "Unregister"

4. 重新載入頁面


方法 C: 使用 Incognito Mode（無痕模式）
---------------------------------------
1. Ctrl + Shift + N 打開無痕視窗

2. 訪問：https://192.168.10.100:3000

3. 應該會立即顯示安裝提示

4. 注意：無痕模式安裝的 PWA 在關閉無痕視窗後會消失


方法 D: 直接從 Chrome 菜單安裝
-------------------------------
1. 訪問：https://192.168.10.100:3000

2. Chrome 菜單（右上角三個點）

3. 找到 "安裝 Leadman FWH System..."

4. 點擊安裝

5. 如果沒有這個選項，說明：
   - PWA 可能已經安裝過
   - 或者需要滿足某些條件（多次訪問等）


檢查是否已安裝：
----------------
訪問：https://192.168.10.100:3000/check-installed.html

這個頁面會告訴你：
✅ PWA 是否已安裝
✅ 安裝提示是否可用
✅ 提供手動觸發安裝的按鈕


Chrome 地址欄圖標：
-------------------
如果 PWA 可以安裝，地址欄右側會出現：
⊕ 圖標（加號圖標）

點擊它就可以直接安裝！


驗證安裝成功：
--------------
安裝後：
1. Windows 開始菜單應該出現 "Leadman FWH System"
2. 桌面可能出現快捷方式
3. 從開始菜單啟動，應該在獨立窗口中打開（無地址欄）
4. 再次訪問 check-installed.html，應該顯示 "PWA 已安裝並從主屏幕啟動"
